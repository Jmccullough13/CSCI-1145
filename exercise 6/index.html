The flexbox layout is used to lay out the children of a parent container in any direction while being able to change which direction the children elements are oriented. It can "flex" the sizes of the element, growing or shrinking if necessary. After being set to flex, the flex property itself can be modified to dictate how the elements can be arranged or how much space each can take up together or individually.

The grid layout is used to display elements in a grid pattern, and can modified to include the number and size of the rows or columns, the space between the elements in the grid, and the individual elements' positions within the grid.

The multiple column layout can be used to change the layout of elements into multiple columns. The attributes of the columns can be changed, such as height, width, or number of columns.
